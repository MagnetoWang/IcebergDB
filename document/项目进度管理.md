### CMakelist

- set
  - 显示指定变量
  - set(var value)
  - var是变量名字一般全大写+下划线
- message
  - 向终端打印信息
  - message(FATAL_ERROR "Folly requires a 64bit target architecture.")
  - FATAL_ERROR 是打印类型
  - 后面字符串是信息
- add_executable
  - 生成执行文件
  - add_executable(test_map ${SOURCE_FILES})
  - test_map就是可以直接运行的文件
- make clean
  - 可以清理工程
